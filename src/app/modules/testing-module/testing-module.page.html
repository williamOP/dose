Metadata

Metadata is automatically pulled from the moduleData.service file when the constructor is called with the associated id.

Metadata is visible in the header and in the information tab.

Footer is clickable to show displayedResultDescription this.metadata.displayedResult: string this.displayedResultDescription: string (html supported) Filters

updateFilter() is called when the filter button is dismissed and so the expected behaviour should be defined under a function of this name in the module.

shownInFilter() returns checks the filter category for a matching filter name and returns true if it is chekced (shown is true)

getSortByProperty() returns the property name of the active sortBy item

To add to filters, use this.addToFilter(); to add to sortByList, use addToSortByList()

To sort an array alphabetically by a property, use this.sortArrayByProperty()

Search button/bar

A search button is shown when this.searchbarPlaceholder is defined. It will also use that value as the searchbar's placeholder. The text will be stored in this.searchbarText. updateFilter() is run when input changes.

this.searchbarPlaceholder: string
